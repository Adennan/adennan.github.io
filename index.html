    golang 交叉编译与 docker 部署

    交叉编译 交叉编译简单地说是一个平台生成另一个平台的可执行程序。 下面先在 Windows 下编译为 Linux 程序 # build.bat SET CGO_ENABLED = 0 SET GOOS = linux SET GOARCH = amd64 go build -o out/linux/gin_hello # 编译后的文件输出到 out/linux下 exit 你可以在项目的根目录下新建立一个 build.bat 然后执行该文件。makefile 安装 Docker OS: centos7 更新软件源：sudo yum update 添加 docker 软件源：sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo 再次更新软件源：sudo yum update 安装 docker ：sudo yum install docker-ce 添加到开机启动：sudo systemctl enable docker 启动 docker ：sudo systemctl start docker 为 Docker 设置远程连接 以下环节是使得 docker 暴漏在公网环境中。...
